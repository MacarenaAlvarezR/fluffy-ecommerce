# 🛍️ Fluffy - Catálogo Responsive

Proyecto de sitio web tipo catálogo de productos desarrollado con **HTML y CSS**, enfocado en diseño responsive usando **CSS Grid, Flexbox y Media Queries**.

---

## ✨ Características

- 📱 Diseño responsive (mobile-first)
- 🧭 Layout con CSS Grid
- 🎯 Uso de Flexbox para alineación de elementos
- 🍔 Menú hamburguesa en versión móvil (UI)
- 🛍️ Catálogo de productos tipo tienda online
- 🎨 Estilos personalizados

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- Media Queries
- CSS Grid
- Flexbox
- Bootstrap (solo componentes)
- Font Awesome (iconos)
